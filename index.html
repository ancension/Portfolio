Bullkit

Bullkit is a mobile application developed using Flutter, designed to deliver a seamless and efficient user experience across both Android and iOS platforms. As the lead programmer, I was responsible for the complete development of the app, from architecture design to feature implementation and optimization. Bullkit combines a clean user interface with powerful functionality, demonstrating the flexibility and performance benefits of cross-platform development using Flutter.

SZN Couture

I designed and developed three separate websites for SZN Couture—a sleek, modern online fashion boutique that empowers users to redefine elegance with every outfit. These include a client-facing website for shopping curated seasonal collections, a rider website for managing deliveries, and an admin dashboard for overseeing inventory and operations. Together, these platforms deliver a seamless fashion experience—so customers can wear their confidence, anytime, anywhere.

Overlord
